# 🎨 童梦绘本工厂

一个帮助宝妈选择和生成个性化定制绘本的网站。

## ✨ 功能特点

- 🌈 可爱的绘本风格界面设计
- 🌞🌙 支持日间/夜间模式切换
- 📚 四个主要页面：首页、登录注册、个性化表单、绘本展示
- 🎨 3D艺术字效果和精美动画
- 📖 逼真的翻书动画效果
- ⚙️ 手动/自动翻页模式
- ✏️ 绘本内容编辑功能
- 💬 用户反馈系统

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📱 页面说明

### 1. 首页 (/)
- 展示网站标题和宣传标语
- 3D艺术字效果
- 特色功能介绍

### 2. 登录注册页 (/auth)
- 用户登录
- 新用户注册
- 可爱的卡通装饰

### 3. 个性化表单页 (/custom)
- 收集孩子基本信息（姓名、年龄、性别）
- 选择兴趣主题和教育目标
- 自定义特殊要求
- 实时进度显示

### 4. 绘本展示页 (/book)
- 逼真的翻书效果
- 手动/自动翻页切换
- 三种翻页速度（慢、一般、快）
- 编辑绘本内容
- 用户反馈功能
- 重新开始阅读

## 🎨 技术栈

- Vue 3
- Vue Router
- Vite
- CSS3 动画

## 🌟 特色设计

- 可爱的卡通角色
- 流畅的页面过渡动画
- 响应式设计，支持移动端
- 童趣风格的配色方案
